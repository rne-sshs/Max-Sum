### Max-Sum Algorithm
Max-Sum Belief Propagation: Factor Graph에서 노드 간 메세지를 통해 목적 함수의 합을 최대화하는 알고리즘.
이를 Group Blind Date 상황에 적용 -> 모두의 만족도 합을 최대화하는 것이 목적.
